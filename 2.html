Listen, Tupel, Sets und Dictionaries

 

Komplexe Datenstrukturen

7

Mit diesen Datenstrukturen lässt sich eine Sammlung vieler Werte speichern:

  • list (Liste): ListeSortiert, Zugriff über Positions-Index (0, 1, 2, ...)
  • tuple (Tupel): simple unveränderliche Liste
  • set (Menge): Ungeordnet, keine Duplikate
  • dict (Dictionary): Menge von Schlüssel-Wert-Paaren;
    Schlüssel-basierter Zugriff
Listen haben eine Ordnung, d. h. es gibt ein nulltes Element, ein erstes und so weiter. In Listen darf ein Element auch doppelt vorkommen. Mengen (Sets) sind dagegen ungeordnet und frei von Duplikaten. Fügt man ein Element in eine Menge ein, welches schon vorhanden ist, passiert nichts.
Dictionaries heißen in anderen Programmiersprachen Maps oder assoziative Arrays. Sie sind eine ungeordnete Menge von Schlüssel-Wert-Paaren. Man greift auf einen Wert zu, indem man den Schlüssel angibt.

Listen

7.1
  • Eckige Klammern: []
  • Geordnet (jedes Element hat seine Position, beginnend mit 0),
  • Duplikate (doppelte Werte) sind möglich.
Eine Liste wird erzeugt mit eckigen Klammern ([] ist eine leere Liste), in der ihre Elemente Komma-getrennt aufgezählt werden. Über eckige Klammern greift man lesend und überschreibend auf die einzelnen Elemente zu. len liefert die Anzahl der Elemente.

List Comprehension

7.1, 8.3

neue_liste = [... for ... in ... if ...]

  • Der Ausdruck wird auf jedes Element angewendet.
  • Hinter for gibt man dem Element einen Namen.
  • Nach in steht ein Iterable (z. B. eine Liste, Menge, Tupel, Dictionary, ...)
  • Danach kann optional noch ein if und eine Filter-Bedingung stehen.

Welche Elemente sind in der Liste res?

Mengenoperationen

7.4
  • a | b Vereinigung (die Elemente aus a und die Elemente aus b)
  • a & b Schnittmenge (die Elemente, die sowohl in a als auch in b sind)
  • a - b Differenz (die Elemente, die in a, aber nicht in b sind)

Zusammenfassung

  • Listen: [], len, append, sum, ...
  • List Comprehension: [... for ... in ... if ...]
  • Tupel: ()
  • Packen und Entpacken: *
  • Mengen: {}, add, x in s, &, |, -, ...
  • Dictionaries: {k:v}
  • for e in liste|tupel|set|dict: ...
